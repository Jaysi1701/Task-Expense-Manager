# Task Expense Manager

A responsive web application built using HTML, CSS, and JavaScript to manage daily tasks and track expenses efficiently.

## Features

* Login Form Validation
* Task Manager (Add, Delete, Search Tasks)
* Expense Manager with Total Calculation
* Dashboard Analytics
* Motivational Quotes using Fetch API
* Dark Mode Toggle
* Local Storage Persistence
* Responsive Design

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Fetch API
* Local Storage

## Concepts Covered

* HTML Fundamentals & Semantic HTML
* Forms and Validation
* CSS Basics, Box Model
* Flexbox & Grid
* Responsive Design
* Arrays, Objects, Functions
* ES6 Classes
* DOM Manipulation
* Event Handling
* Promises & Async/Await
* Browser APIs
* Fetch API
* Local Storage
